# Yesilay Media

Premium landing page for AI-powered media & communication company.

## View

Open `index.html` in your browser.

Or use a local server:
```bash
python3 -m http.server 3001
```
Then visit: http://localhost:3001

## Customize

Edit `index.html`:
- Line 140-141: Team names
- Line 166: Email address
- Line 173-175: Social links

## Design

- Pure black & white (#000000, #FFFFFF)
- Inter font family
- No gradients, no colors
- Premium minimal aesthetic
- Mobile responsive
